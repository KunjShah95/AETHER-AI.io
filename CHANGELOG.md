### Development Setup
```bash
# Clone the repository
git clone https://github.com/KunjShah95/NEXUS-AI.io.git
cd NEXUS-AI.io

# Install dependencies
cd terminal
pip install -r requirements.txt

# Set up environment
cp .env.example .env
# Edit .env with your API keys

# Run the application
python main.py
```
