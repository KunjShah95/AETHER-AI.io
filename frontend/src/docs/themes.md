# UI Themes

Customize the look and feel of your terminal with built-in themes or create your own.

## Built-in Themes

| Theme | Command | Description |
|-------|---------|-------------|
| **Default** | `/theme default` | Classic dark theme |
| **Neon** | `/theme neon` | Bright neon colors |
| **Ocean** | `/theme ocean` | Ocean blue theme |
| **Forest** | `/theme forest` | Forest green theme |
| **Sunset** | `/theme sunset` | Warm sunset colors |
| **Minimal** | `/theme minimal` | Clean minimal design |

## Custom Themes

You can create and manage custom themes.

- **Create**: `/theme create [name]`
- **Set Property**: `/theme set [property] [value]`
- **Save**: `/theme save [name]`
- **Load**: `/theme load [name]`
- **Delete**: `/theme delete [name]`
- **List**: `/theme list`

### Theme Properties

- `primary_color`
- `secondary_color`
- `background_color`
- `text_color`
- `border_color`
- `success_color`
- `error_color`
- `warning_color`

## Management

- **Preview**: `/theme preview [name]`
- **Export**: `/theme export [name]`
- **Import**: `/theme import [file]`
- **Reset**: `/theme reset`
- **Info**: `/theme info [name]`
